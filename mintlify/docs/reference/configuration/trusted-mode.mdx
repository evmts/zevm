## Mining Rules

| Mining type | Required fields | Invalid fields |
| --- | --- | --- |
| `auto` | none | `blockTime` |
| `manual` | none | `blockTime` |
| `interval` | `blockTime` | none |

## Fork Rules

- `fork.blockNumber` is invalid without `fork.url`
- forking stays inside trusted mode; it does not create a third mode
- `chainId` remains a local trusted-mode setting and is not inferred from `fork.url`
