# Canonical Queries

<CurrentHeadNote />

## Methods

| Method | Contract |
| --- | --- |
| `eth_getBlockByNumber` | return a canonical trusted-mode block by block tag or number |
| `eth_getBlockByHash` | return a canonical trusted-mode block by hash |
| `eth_getTransactionByHash` | return a canonical committed transaction |
| `eth_getTransactionReceipt` | return the canonical receipt for a committed transaction |
| `eth_getBlockReceipts` | return all canonical receipts for a block |
| `eth_getLogs` | return canonical logs from trusted local chain history |

## Block Tags

For tag-taking canonical queries in trusted mode:

- `latest` = current local head
- `pending` = alias of `latest`
- `safe` = alias of `latest`
- `finalized` = alias of `latest`
- `earliest` = block `0`

Those aliases do not provide real finality.
