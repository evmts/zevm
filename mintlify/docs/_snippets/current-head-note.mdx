> Contract first, `HEAD` second: this page defines the intended ZEVM product contract. As of 2026-03-29, current `HEAD` does not satisfy that contract: `zig build` fails first in `src/rpc/server.zig` because current upstream `jsonrpc` no longer exposes `jsonrpc.envelope`, `zig build test` fails in `src/rpc/dispatcher_test.zig`, `src/rpc/server.zig`, and `src/tx_processor.zig` because current upstream `jsonrpc` no longer exposes `jsonrpc.envelope` and current `guillotine-mini` requires a wider `evm.init` signature, `src/main.zig` still only wires `src/rpc/server.zig.parseConfig`, `parseConfig` still only accepts `--host` and `--port`, and the live listener still starts with an empty handler registry.
