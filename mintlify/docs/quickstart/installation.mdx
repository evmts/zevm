# Installation

ZEVM installation is source-build only in the phase-1 public contract.

The settled install path is:

1. Check out ZEVM in a workspace that also contains sibling `../voltaire` and `../guillotine-mini` repositories, because `build.zig.zon` uses local path dependencies.
2. Run `zig build` from the ZEVM repository root.
3. Use `./zig-out/bin/zevm` from the build output.

No packaged binary, installer, or package-manager path is part of the public phase-1 stance.

## Source-Build Path

```bash
zig build
```

The binary name is `zevm`. This page defines the intended source-build contract, not a claim that current `HEAD` produces a runnable executable.
